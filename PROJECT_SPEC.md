# tomuss.com — プロジェクト仕様書

## 概要
株式会社須藤商店の企業サイト。1976年創業、秋田県にかほ市の地域密着型総合生活サービス企業。
取引先・お客さん両方が見るサイト。信頼感＋美しさを両立させる。

## コンセプト
**「みなさまの生活を豊かにする」**
地域のインフラを支える企業としての信頼感と、洗練されたデザインの両立。

## デザイン仕様

### 全体のトーン
- 企業としての信頼感・安心感が第一
- でも「うおー」ってなる美しさも
- ネイビー×ゴールドで品格を出す
- 写真は実際の店舗写真を使用

### サブページのテンプレート構造
各サブページは統一されたレイアウト：
1. ページヒーロー（部門名＋写真、高さ控えめ）
2. コンテンツ本文
3. 店舗情報（住所・電話・地図）
4. 共通フッター

### 写真素材
- 現サイトの写真を使用：
  - ガソリンスタンド夕焼け
  - くるま屋トム外観
  - トム太商店外観
  - 本社外観
- 住宅設備・農園部の写真は後から差し替え
- 各サブページの詳細な写真は追加可能

### レスポンシブ対応
- トップのカードグリッド：PC3列→スマホ1列
- フッターナビ：PC3列→スマホ2列
- フリーダイヤルはスマホでタップして電話できるように
